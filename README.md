### *Bringing agricultural and climate data together for better insights.*

AgriInsight is an interactive web-based data visualization tool designed to combine agricultural production and rainfall data from **data.gov.in**.  
It helps policymakers, researchers, and citizens explore state-wise trends and analyze the correlation between crop yields and rainfall patterns in India.
